就那烟囱效应就能说这结果是奸商跟保护伞一个「利益最大化」一个「不作为」的妥妥人祸。

至于吵架，你港内两地不吵起来，你们好好辩论，万一扯到「既然有非遗阻燃竹那么此次的为何不合标准你们怎么监管这标准」的话题上，更多保护伞是不是就把持不住了？所以我在看到一帮媒体一口咬定主要问题在竹的时候我就觉得对面的大奸大恶在求生欲上还是挺耐力的，两头用力去构建针锋相对的舆情、还是精挑细选了话题使得这个架是那种只会越吵越烈却吵不出结果的，两边儿群众斗群众然后港地大佬们就能坐壁上观安得很。

就这完美的助燃结构，我觉得起火原因都别找了，这里的居民一定十分甚至万分地小心然后居民的祖宗估计也在下面把关系找遍了，才有的今天这般，工期过了这么久才烧起来，这天王老子来了都得烧高香啊不是？而且就凭能做出这种结构来一点，你就没法相信它会在别的地方注意安全，阻燃材料或者竹子性能是否合格标准怎么执行？不存在的，都不存在的，它能给你安一个水龙头这种现象都能存在的时候港那边的大佬就已经手上沾满了住户的血了。住户没立即身亡，那是住户的本事，奸商们修要偷天之功，好吧。

而且，工期搞这么久，我只能说它们修的不是自己的楼它们是真不着急哈。

这里的根本问题在于，即便他们换钢架了，但如果他们那边的商业利益最大化最大受益方没换，那还是没解决问题。再高的建筑我相信也有办法设计足够合理的结构，从而在即便使用竹子的前提下不会出现包括燃烧或易折在内的任何问题。但问题是，它已经让不该易燃的不为易燃甚至在结构上还促燃了，就像我不信它还会好好搞竹架的炮制工序一样，我也不相信它会好好关心避免易折的设计。何况竹坚固度的设计可能考虑的事情要远多于钢材的坚固度设计 …… 它能整出来这种硬活，我是断然不相信钢结构它能愿意用心设计，那竹就更不用提了，易折是板上钉钉的事，但这事儿可不是换个材质就能解决的，恰恰是竹子暴露了竹子以外的问题。

所以我觉得，当看到大家说竹子材料的性能问题的时候，就可以这样子来理解。讲性能问题，不是说换另一种性能的材料就能解决问题，而是说，你看它连最基本的问题都不上心的前提下还让它高层用竹，能够默许这种状况的背后财团泯灭人性到简直令人发指！它不是保护非遗它是在耗尽非遗的名声来给自己最后谋求一丝丝的可压榨利润啊！

就这等奸商德性，换钢架照样给你烧化咯。它就没在意过这儿的人的性命，它兴许都为了利润自己都不怕死了、它出卖别人性命的时候能多眨一下眼吗？幻想「对方还是个人」这种事儿没意义，就好比在这次事件中咬定竹为主因一样没意义，也像是只讨论竹是非遗性能很好却不提这次事件里的竹架是否也有相同性能一样没意义，也如同幻想只要换掉竹就能万事大吉一样没意义。咱们要强调一个基本生命常识都没有的开发方都能来做事了，结果一个个就竹子吵起来了仿佛替换了竹子就能解决问题似的？仿佛竹架是非遗手艺就没事儿似的？死人了啊！两拨人搁这儿讲笑话玩儿呢？
