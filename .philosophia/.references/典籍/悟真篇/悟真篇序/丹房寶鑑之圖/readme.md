懸胎鼎：
~~~
鼎周圍一尺五寸，中虛五寸，長一尺二寸。
狀似蓬壺，亦如人之身形。分三層應三才。
鼎身腹通直，令上中下等均匀，入鑪八寸，
懸於𥩋中不著地。懸胎是也。又謂之朱砂鼎，
張隨注云：又名太一神爐。
~~~

鐵牛：
~~~
滿庭芳
------

真鐵牛兒，形容醜惡，性剛偏好爭馳。
人人皆有，那箇解牽騎。
種就黃芽滿院，更須用、神水澆之。
中宮里，若無此獸，安得似嬰兒。

乾坤，真動靜，生成家活，總賴於伊，
飢飡虎髓，渴飲水銀池。
夜半牽車進火，霞光道，海底騰輝。
牧童笑，華池宴罷，乘箇月明歸。
~~~

偃月鑪：
~~~
鑪面周圍約一尺二寸，明心，橫有一尺，立唇，
環匝二寸，唇厚二寸，鑪口偃開鍋釜，又如仰月狀，
故名偃月鑪也。張隨注云：又名威光鼎也。
~~~
